Introductory Programmers

This project aims to enrich the curricula of online educational programming platforms in order to convince students that studying computer science is important, relevant, and related to their day-to-day experiences.

Domain-Experts

Furthermore, DataSnap is not just designed for educational contexts. The platform can enable domain experts who commonly possess little programming expertise to both analyze data and visualize their results.

Data will add the following to the original Snap! platform:

  • real-time and big-data libraries into the Snap! platform
  • big-data import from a user's private data sets
  • the use of computational resources (cloud servers) to speed up the processing of big data
  • combining third-party visualization libraries to provide domain experts and other users with easy-to-use interfaces to visualize their results

Real-Time and Big-Data Libraries:

  • Weather Data
  • Google Finance Data
  • Twitter Data
  • Reddit Data
  • Google GeoCoding Data
  • USGS Earthquakes Data


Enriching Online Educational Programming Platform Curricula

One of the primary aims of this project is the enrich the curricula of online educational programming platforms.

This project will provide online programming platforms the following useful contexts in which to study computer science:


Contexts:

  • Learning computer science concepts
    • Ex: Learning the concepts of calculation, sequence, decision, and iteration while using relevant data.
    • Ex: Learning how to create and use methods or functions.
  • Social media data analysis
    • Ex: Using Twitter to predict which upcoming movies will sell the most tickets.
    • Ex: Using Reddit to determine which news stories are the most popular.
  • Visualizing Data
    • Ex: Using Google Maps to visualize the USGS reported earthquakes.
    • Ex: Using Google Maps to visualize which airports have inclement or dangerous weather conditions.
  • Streaming Data
    • Ex: Streaming stock market data to determine which stocks are currently up or down for the day.
    • Ex: Streaming stock market data to analyze a personal stock portfolio.

Abstract


Online educational platforms (e.g., Massive Open Online Courses (MOOCs)) have become increasingly popular means of delivering computing curricula. These platforms, unfortunately, are plagued by extremely low retention rates, particularly in introductory courses. Educational theory posits that improper academic motivation models can be responsible for large percentages of online students never completing their courses. Specifically, the current focus of online programming assignments on entertainment (e.g., games, animation, etc.) is particularly misdirected; it fails to convince students that the covered technical content is important, relevant, and related to their day-to-day computing experiences.


As an alternative, this project will focus on real-time and big datasets as a means of convincing introductory online learners that computing is a powerful tool for solving challenging and relevant problems across a variety of fields. The specific goal of the proposed project is to address the challenges of enriching the curricula of online educational programming platforms with real-time and big data sources by creating the necessary scaffolding and programming abstractions. The resulting methodology will enable other online educational platforms to improve their motivation and retention rates by leveraging our technology to enrich their curricula.
